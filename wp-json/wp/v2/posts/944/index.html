CVE-2023-35082 es una vulnerabilidad cr\u00edtica que permite acceder a las API en versiones antiguas de MobileIron Core<\/p>\n<\/blockquote>\n\n\n\n

Ivanti no levanta cabeza, tras la \u00faltima vulnerabilidad cr\u00edtica, un nuevo hallazgo ha sido reportado. Se trata de la vulnerabilidad CVE-2023-35082<\/strong>, que afecta<\/a> a versiones m\u00e1s antiguas y no soportadas de MobileIron Core<\/strong>.<\/p>\n\n\n\n

MobileIron Core es un producto descontinuado que permite a las organizaciones gestionar dispositivos m\u00f3viles<\/strong>, como tel\u00e9fonos y tablets.<\/p>\n\n\n\n

La vulnerabilidad CVE-2023-35082<\/strong> permite a atacantes no autenticados acceder a las API en versiones antiguas de MobileIron Core (11.2 y anteriores). Esto significa que un ciberdelincuente podr\u00eda acceder a los endpoints de la API en el servidor de gesti\u00f3n expuesto sin necesidad de autenticaci\u00f3n<\/strong>. Con este acceso, un atacante tendr\u00eda la capacidad de acceder a datos personales o realizar modificaciones en la plataforma. Adem\u00e1s, el atacante podr\u00eda encadenar<\/a> la vulnerabilidad CVE-2023-35081<\/strong>, aumentando el riesgo y la gravedad del ataque.<\/p>\n\n\n\n

Caracter\u00edsticas principales<\/h2>\n\n\n\n

Las caracter\u00edsticas principales de la vulnerabilidad CVE-2023-35082<\/strong> son:<\/p>\n\n\n\n

    \n
  • Identificador CVE<\/strong>: CVE-2023-35082<\/li>\n\n\n\n
  • Valor CVSS<\/strong>: 10.0 (AV:N\/AC:L\/PR:N\/UI:N\/S:C\/C:H\/I:H\/A:H)<\/li>\n\n\n\n
  • Fecha de publicaci\u00f3n<\/strong>: 02\/08\/2023<\/li>\n\n\n\n
  • Software afectado<\/strong>: MobileIron Core<\/li>\n\n\n\n
  • Versiones afectadas<\/strong>:\n
      \n
    • MobileIron Core \u2013 11.2 y anteriores<\/li>\n<\/ul>\n<\/li>\n<\/ul>\n\n\n\n

      Es importante tener en cuenta que esta vulnerabilidad se deriva de un problema de naturaleza permisiva de ciertas entradas en la cadena de filtros de seguridad de la aplicaci\u00f3n web. Esto la convierte en evasi\u00f3n de otra vulnerabilidad, la CVE-2023-35078<\/strong>, en las versiones 11.2 y anteriores del producto.<\/p>\n\n\n\n

      Mitigaci\u00f3n y recomendaciones<\/h2>\n\n\n\n

      Ivanti, ha emitido un comunicado al respecto. Sin embargo, debido a que MobileIron Core 11.2 ya no recibe soporte desde el 15 de marzo de 2022, no se proporcionar\u00e1 un parche para esta vulnerabilidad ni para versiones anteriores. Por lo tanto, la mejor soluci\u00f3n es migrar a la \u00faltima versi\u00f3n de Ivanti Endpoint Manager Mobile<\/strong> (EPMM).<\/p>\n\n\n\n
